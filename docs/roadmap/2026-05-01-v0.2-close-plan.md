## What this doc is

The v0.2-to-v1.0 roadmap is the strategic framing. This doc is the
operational decomposition — what concrete Bursts close v0.2, what
order, what's earned at each. It's read alongside the roadmap, not
in place of it.

The v0.2 close target is a **deliberate, audit-traceable** v0.2.0
release that captures:
- The SarahR1 absorption arc (already shipped in v0.1.2)
- The remaining ADR-0021-am tool annotations
- ADR-0035 / ADR-0036 / ADR-0037 minimum-bar implementations
- The Phase G.1.A programming-primitives batch (per the roadmap)
- A clean STATE refresh + CHANGELOG entry

This is **3-4 weeks of work at sustained pace** if all the v0.3
ADR implementations land for v0.2. **OR** v0.2 ships with the
v0.3 ADRs Proposed-but-not-implemented and Phase G.1.A as the
substantive feature work. The latter is what this plan defaults to —
ADR-0035/0036/0037 implementation is genuinely v0.3 work; v0.2
should focus on closing the existing absorption surface + Phase
G.1.A.

## Sequencing recommendation

**Path A — v0.2 with all the prep groundwork (recommended).**
Order: 49 → 50 → 51 → 52 → 53 → 54 → … → 62 → 63 → 64. Each Burst
ships with full test coverage + green suite + commit + push. The
v0.3 ADR prep work (50-52) lands first because it's small and
unblocks future tranches; Phase G.1.A is the substantive feature
work (53-62).

**Path B — v0.2 minimal (skip v0.3 prep).**
Order: 49 → 53 → 54 → … → 62 → 63 → 64. Drop 50/51/52; v0.2
ships with ADR-0035/0036/0037 still entirely Proposed. Faster to
ship; the v0.3 ADRs need their own absorption arc later.

**Path C — Defer Phase G to v0.3, ship v0.2 as paperwork only.**
Order: 49 → 63 → 64. v0.2 captures the SarahR1 absorption arc +
per-tool annotations (Burst 49). No new feature work. Cleanest
shipping path; v0.2 is genuinely "the SarahR1-absorption release"
nothing more.

The orchestrator (Alex) chooses. **Default lean: Path B** — Phase
G.1.A is meaningful feature work (operator-visible: SW-track
agents can lint, run tests, read git history), and v0.3 ADR
implementation is properly its own absorption arc.
